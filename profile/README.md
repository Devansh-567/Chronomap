# Chronomap

Tools for working with time-versioned data in Python. Started as a single
project, split into a few focused repos as it grew:

| Repo | What it is |
|---|---|
| [`Chronomap`](https://github.com/Devansh-567/Chronomap) | The core library — a thread-safe, time-versioned key-value store. |
| `chronomap-examples` | Runnable, standalone examples (config store, session store, audit log, etc.) — planned |
| `chronomap-benchmarks` | Reproducible performance benchmarks with real numbers, not README claims — planned |

No pressure, no roadmap theater — this is a small project maintained in
spare time. Issues and PRs are read and responded to, just not always
quickly.
