# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1.

In short: be respectful, assume good faith, and keep disagreements about
code, not people. Harassment, personal attacks, and discriminatory
language or behavior aren't tolerated.

## Reporting

If something happens that violates this, email the maintainer at
devansh.jay.singh@gmail.com. Reports are handled privately.
